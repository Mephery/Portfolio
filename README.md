## DE LA SYNTAXE AUX SYSTÈMES

Anciennement prof de lettres, j'ai troqué l'analyse des structures textuelles pour la rigueur de la logique binaire. Actuellement en **BTS SIO en alternance**, je décompose les systèmes, builds des architectures denses et sécurise les réseaux avec la même exigence.

## LA STACK

### 3D & Frontend
*   **Astro & React** : Architecture insulaire performante pour un chargement foudroyant.
*   **React Three Fiber (Three.js)** : Gestion d'un champ unifié de **100 000 particules** synchronisées.
*   **Custom GLSL Shaders** : Calculs mathématiques déportés sur le GPU pour le morphing fluide des 7 nœuds géométriques au scroll (Cortex avec bruit harmonique, Attracteur de Lorenz chaotique, etc.).

## KEY ARCHITECTURES SHOWN

*   **Node 01 // À Propos** : Génération procédurale d'un avatar et de sa station de travail en point-cloud (MeshSurfaceSampler).
*   **Node 02 // L'Entreprise** : Trois engrenages mathématiques imbriqués avec rotation asynchrone calculée par le Vertex Shader.
*   **Node 03 // Chaos** : Modélisation physique de l'Attracteur de Lorenz pour introduire mon application de chat sécurisé.
*   **Node 04 // Compétences** : Ellipsoïde 3D asymétrique sculpté par ondes de fréquences sinus/cosinus pour simuler un cortex cérébral organique.
